# Thyroid Tracker

Local-first Flutter app for hypothyroidism tracking: labs, medication intake, doctor visits, medical media, and reference notes.

## Start

Flutter SDK is not currently available in this shell PATH. After installing Flutter or adding it to PATH, run:

```powershell
flutter create --platforms=android,ios,windows,web .
flutter pub get
flutter analyze
flutter run
```

The app source is already placed in `lib/`.
